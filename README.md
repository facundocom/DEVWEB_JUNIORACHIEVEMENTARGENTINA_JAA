=====================================================
 DEVWEB - JUNIOR ACHIEVEMENT ARGENTINA (JAA)
=====================================================

DESCRIPCIÓN DEL PROYECTO
-----------------------------------------------------
Este repositorio contiene el desarrollo de un sitio web simple, creado como proyecto final o ejercicio práctico para un curso de la organización Junior Achievement Argentina (JAA).

El proyecto fue desarrollado como parte de un "little course" (pequeño curso) de JAA, con el objetivo de aplicar los conocimientos fundamentales de desarrollo web.

CARACTERÍSTICAS
-----------------------------------------------------
* **Sitio Web Estático:** Es un proyecto de front-end simple que no requiere base de datos o servidor.
* **Diseño Responsivo:** Se busca que sea visible correctamente en diferentes dispositivos.
* **Interacción Básica:** Utilización de JavaScript para cualquier funcionalidad dinámica.

TECNOLOGÍAS UTILIZADAS
-----------------------------------------------------
* **HTML5:** Estructura principal del sitio web.
* **CSS3:** Estilos visuales del sitio (definidos en `style.css`).
* **JavaScript (JS):** Lógica y funcionalidades interactivas (definidas en `main.js`).

ESTRUCTURA DEL REPOSITORIO
-----------------------------------------------------
El proyecto está organizado de la siguiente manera:

* **/** (Raíz del proyecto)
    * **FACUNDO_GABRIEL_CORIA_HADDAD/**
        * `index.html`: El archivo principal del sitio web (página de inicio).
        * `style.css`: La hoja de estilos para la presentación visual.
        * `main.js`: Archivo que contiene el código JavaScript.
        * `imagenes/`: Directorio que contiene todos los recursos visuales (logos, fotos, etc.).

CÓMO VISUALIZAR EL SITIO
-----------------------------------------------------
Dado que es un proyecto de desarrollo web estático, puedes visualizarlo de forma inmediata:

1.  Abre el explorador de archivos y navega hasta la carpeta: `FACUNDO_GABRIEL_CORIA_HADDAD/`
2.  Haz doble clic en el archivo `index.html`.
3.  El sitio se abrirá automáticamente en tu navegador web predeterminado.
